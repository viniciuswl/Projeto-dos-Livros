# **Projeto dos Livros**
## Sistema de armazenamento de dados livros usando Python.

## **Front-end**
### Tecnologias utilizadas:
- Python
- biblioteca Tkinter

### Features:
* Capacidade de criar, alterar, deletar e buscar livros registrados no banco de dados do programa
